# Unity PvE 遊戲 Demo - 使用 Cursor開發

## 項目介紹
這是一個由遊戲開發學生開發的簡單 PvE 遊戲原型，使用 Unity 引擎與 C# 實現。

## AI 輔助開發過程
- **工具**：Cursor AI
- **使用方式**：透過自然語言描述功能，讓 AI 生成 MonoBehaviour 腳本，並進行迭代 debug。
- **成果**：大幅縮短腳本撰寫與除錯時間，快速完成可玩 Demo。

## 主要功能
- 玩家移動、跳躍、衝刺系統
- 敵人 AI 狀態機（巡邏、追擊、攻擊）
- 基礎戰鬥與血量系統
- 物品拾取機制
- 簡單關卡流程

## 運行方式
1. 下載專案
2. 用 Unity 6000.3.6f1 或更高版本打開 `UnityProject` 資料夾
3. 運行場景 `MainScene`
